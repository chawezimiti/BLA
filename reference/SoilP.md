# Soil Phosphorus data

This data set is a subset of a data set that was assembled by AgSpace
Agriculture Ltd in a soil survey conducted on farms in various
management units across England.

## Usage

``` r
SoilP
```

## Format

A data.frame with 6020 rows and 2 columns:

- yield:

  Wheat yield (ton/ha) measured

- P:

  Phosphorus (mg/l)

## Examples

``` r
data(SoilP)
```
